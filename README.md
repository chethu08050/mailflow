# MailFlow Pro - Email Marketing Platform

A modern, responsive website for MailFlow Pro - a bulk email sending and communication automation tool.

## Project Overview

This project implements a complete website for MailFlow Pro with:

1. **Homepage** - Featuring hero section, features, how it works, pricing, testimonials, and call-to-action
2. **Dashboard** - Interactive dashboard UI showing email campaigns, analytics charts, and contact lists
3. **Contact Page** - Contact form, information, and FAQ section

## Design Features

- **Modern SaaS Design** with blue-white theme
- **Fully Responsive** layout that works on mobile, tablet, and desktop
- **Rounded UI Components** with subtle gradients and smooth animations
- **Dark Mode** support with toggle switch
- **Card-based Layout** for content organization
- **Interactive Elements** including accordions, toggles, and hover effects

## How to Use

1. Clone or download the repository
2. Open `index.html` in a web browser
3. Navigate through the site using the navigation menu
4. Toggle dark mode using the button in the bottom right corner

## Customization

To customize the design:
1. Modify color variables in `:root` of `styles.css`
2. Update font families in the `--font-primary` and `--font-secondary` variables
3. Adjust spacing using the spacing variables
4. Modify component styles as needed
